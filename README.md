🍽️ Restaurant & Mentorship Web UI
📌 Project Overview

This project is a responsive web application that includes:

🍴 Rooftop Restaurant Landing Page
👨‍🏫 Mentorship / Team Showcase Page

It is designed using HTML, CSS, and Bootstrap, focusing on clean UI, responsiveness, and modern layout design.

🍴 Restaurant Page
Beautiful rooftop restaurant layout
Image + content side-by-side design
Call-to-action buttons (Menu & Reserve)
Clean and modern UI
Responsive using Bootstrap
👨‍🏫 Mentorship Page
Mentor lead highlight section
Mentor cards with roles & descriptions
Responsive grid layout
Navigation bar with mobile toggle

HTML5
CSS3
Bootstrap 4
JavaScript (basic navigation)
